# AgroSphere 🌱

AgroSphere is a smart agriculture platform developed during the Skill India Hackathon. The platform helps farmers access crop-related information, government schemes, disease detection, weather insights, and market details through a simple and user-friendly interface.

## Features

* 🌾 Crop Information and Advisory
* 🤖 AI-powered Crop Disease Detection
* 💊 Medicine and Treatment Suggestions
* 📈 Market Price Information
* 🌦 Weather Updates
* 🏛 Government Schemes and Subsidies
* 👨‍🌾 Farmer-Friendly User Interface

## Technology Stack

* HTML
* CSS
* JavaScript
* Gemini API
